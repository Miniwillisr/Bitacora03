Pseudoelementos

Ideas Principales

Nos permite insertar contenido en una pagina CSS (Sin que tenga que estar en html)

Apuntes

  • After

    • Inserta algo despues de cada uno de los elementos seleccionados

  • Before

    • Inserta aldo antes de cada uno de los elementos seleccionados

  • Content

    • Se puede usar en cadena de caracteres (content:texto), Imagenes(//url(image.jpg)) y nada. (Util para figuras)

Resumen

Los Pseudoelementos son muy utiles ya que nos permitan una gran versatilidad ya sea para el texto, tanto como, el poder crear figuras utilizando esta tecnica de una forma mas avanzada
