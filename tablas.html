Tablas

Ideas Principales

Nos permite Organizar datos en filas y columnas.

Permite buscar con rapidez y facilidad valores entre diferentes tipos de datos.

Apuntes

  • Thead: Encabezado
  • Th: Celda
  • Tbody: Cuerpo
  • Tr: Renglon
  • td: Celdas
  • Caption: Insertar captions
  • Tfoot: Footer de la tabla
  • Rowspan: Indica el numero de filas que ocupa la celda
  • Colspan: Indica el nunero de columnas que ocupa la celda

Resumen

La etiquta table es muy util dependiendo el tipo de trabajo que queramos realizar, ya que nos permite tener una gran cantidad de datos organizados de una manera coherente y que pueda ser facil de leer para el usuario
