Header

Ideas Principales

Es una de las partes mas importanres de una pagina web, pues es lo que visualiza primero los usuarios, por lo tanto, la cabecera debe ser limpia y ordenada para no llegar a cofundir

Debe de ser un espacio generico que tendra el mismo diseño en todas las paginas web.

    ¿Que elementos debe incluir?

  • Logotipo
  • Menu
  • Buscador
  • Redes Sociales, etc.

    • El unico elemento que no puede faltar es el Logo y este debe enlazar a la pagina de inicio.

Apuntes

  • NAV

    • Puede o no estar dentro del header, depende del proyecto que estemos haciendo. Algunos pueden llegar a necesitar un doble menu

  • Buscador

    • Puede estar en el header por cuestiones de usabilidad. Hay varias opciones de diseño como la lupa

  • Tienda Online

    • Se debe incluir los elementos referentes de compra: carrito, cuenta, pedido, etc.

  • Botones Destacados

    • Sirven para enfatizar un servicio, llevar al usurio o animarle una accion especifica Como pedir presupuesto

  • Barras Fijas

    • Pueden mostrar descuentos, cupones, llanadas a la accion o informacion adicional

  • Selector de idioma

    • Si el proyecto es un sitiomulti lenguaje un elemento obligatorio en el header es un selector de idioma

  • Formulario de Suscripcion

    • No es recomendable, pero se puede ubicar dentro del header Se puden colocar en otras partes como el footer

Resumen

El header es una parte importante de toda pagina web ya que es lo primero que el usuario vera al entrar en nuestra pagina web
